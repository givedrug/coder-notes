# （问题）正则中如何多次匹配字符串中的模式子串？


代码：

```java
Pattern p = Pattern.compile(":[a-zA-Z]+");
Matcher m = p.matcher("This is a :test string with :another example:123");
while (m.find()) {
    System.out.println(m.group());
}
```

输入：

```
This is a :test string with :another example:123
```

输出：

```
:test
:another
```
